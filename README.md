# 📊 Finova – Finance Dashboard

## 📌 Overview

Finova is a clean finance dashboard built to help users track and understand their financial activity. It provides a simple interface to view balances, transactions, and spending insights.

---

## 🚀 Features

### 📈 Dashboard Overview

* Summary cards: Total Balance, Income, Expenses
* Time-based visualization (line chart)
* Basic financial insights

### 📋 Transactions

* Displays transaction details:

  * Date
  * Amount
  * Category
  * Type (Income / Expense)
* Simple table UI

### 👥 Role-Based UI

* Viewer → Can only view data
* Admin → Can access additional controls

### 🧠 Insights

* Highest spending category
* Basic financial observations

## ⚖️ Technical Decisions and Trade-offs

* Used React (Vite) for fast development
* Tailwind CSS for quick and consistent styling
* Recharts for simple chart integration

### Trade-offs:

* Used mock data instead of backend
* Basic state management for simplicity
* Focused on clean UI over complex features
